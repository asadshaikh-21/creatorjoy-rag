# CreatorJoy RAG Chatbot

AI-powered RAG chatbot for comparing social media videos using transcript embeddings and vector search.

## Features

* YouTube + Instagram video processing
* Transcript extraction
* Metadata analysis
* Engagement rate calculation
* RAG-based conversational chat
* ChromaDB vector storage
* Gemini-powered responses

## Tech Stack

* FastAPI
* LangChain
* ChromaDB
* Gemini API
* React (planned)

## Current Progress

* Backend APIs setup
* Vector embedding pipeline integrated
* Gemini integration completed
* Transcript extraction in progress

## Run Backend

```bash
pip install -r requirements.txt
python run.py
```
